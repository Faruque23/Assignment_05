Answer:

  • innerHTML returns the HTML content of an element, including tags.

Answer: Event delegation is a technique where a single event listener is added to a parent element instead of individual child elements. This improves performance and reduces memory usage. The event listener can catch events for all current and future child elements.

Answer: Event bubbling is the process where an event starts from the innermost element and bubbles up to the parent elements in the DOM tree. This allows parent elements to handle events from their children.
